# Docker Desktop
Setup:
+ Install **Docker Desktop**
+ Install **[[WSL]] 2 Backend** by `wsl --install`
This step is necessary since docker tuns natively on UNIX-based systems. This command will install [[Ubuntu]] on your machine.
